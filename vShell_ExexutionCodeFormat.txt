=================================================================================================
📘 vShell - .vex 중간 코드(Bytecode) 32bit x 8 메모리 매핑 마스터 테이블 (v2026.07)
=================================================================================================
* ExecBlock 규격: 32비트 x 8 = 256비트 (32바이트)
* 문자열(String) 인수 처리: 컴파일 시점에 엔진 내부 ID로 Resolve하여 순수 uint32_t ID값으로 Cmd에 삽입. 
  (단, echo 같은 순수 문자열은 Cmd[1]~Cmd[7] 영역에 최대 28바이트(4바이트x7) 텍스트로 직접 패킹 가능)

[메모리 레이아웃 기본 규칙]
- Cmd[0]: [31:24] Opcode (8비트) / [23:16] 조건부 플래그 (8비트) / [15:0] Jump Target 오프셋 (16비트)
- Cmd[1]: 주로 제어 대상 1 (Source Cell ID 등, 32비트)
- Cmd[2]: 주로 제어 대상 2 (Target Cell ID 등, 32비트)
- Cmd[3], Cmd[4]: fixed64_t 데이터 (정수부 32비트, 소수부 32비트)
- Cmd[5], Cmd[6]: 추가 fixed64_t 데이터 또는 보조 인수
- Cmd[7]: 단일 Char 연산자 (op) 또는 기타 플래그 제어용
