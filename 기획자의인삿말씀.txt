1. 대화형 학습 :
   이 시스템의 학습은 두가지 방식으로 진행됩니다. 1. 'A는B다' 형식의 단순 지식은 자연어나 자연어 패컨을 그대로 옮긴 vShell 스크립트를 통해 입력과 학습을 합니다.
   "사과는 무슨색이지?" -> "..." -> "사과는 빨간색이야." -> 내부적으로 '사과' 셀과 '빨간색' 셀, '무슨색' 셀을 만들어 기본 가중치로 서로 연결한다.
   "사과는 무슨색이지?" -> '사과' 셀과 '빨간색' 셀과 '무슨색' 셀이 활성화 된다.

2. 추세와 경향 학습.
	단순 지식의 경우 간단한 셀 설정과 반복 사용에 대해 쉽게 형성이 가능하나 유사한 케이스의 다양한 입력에 대한 추세나 경향을 이해하자면 기존의 블록 레이어 구조의 신경망 형성이 필요합니다.

1. Interactive Learning:

Learning in this system proceeds in two ways. 1. Simple knowledge in the form of "A is B" is input and learned through natural language or vShell scripts that directly transfer natural language patterns.

"What color is an apple?" -> "..." -> "Apples are red." -> Internally, 'Apple', 'Red', and 'What color' cells are created and connected to each other using default weights.

"What color is an apple?" -> The 'Apple', 'Red', and 'What color' cells are activated.

2. Learning Trends and Patterns.

While simple knowledge can be easily formed through simple cell configuration and repetitive use, understanding trends or patterns regarding various inputs of similar cases requires the formation of a neural network with an existing block layer structure. In this regard, this system is designed to enable automatic learning by generating NxNxN neural network blocks as desired, initializing weights randomly, and using user input and expected output patterns as arguments.
