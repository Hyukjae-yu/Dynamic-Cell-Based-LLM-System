📘 vShell 스크립트 및 내장 에디터 통합 마스터 매뉴얼 (v2026.07)
===============================================================================

1. 내장 간이 라인 에디터 (vShell Editor)
-------------------------------------------------------------------------------
* edit <파일명.vsh>          : 에디터 모드 진입 (파일이 있으면 자동 로드)
* list                      : 현재 에디터 버퍼의 내용과 줄 번호 출력
* del <줄번호>               : 지정한 줄의 코드 삭제
* insert <줄번호> <내용>     : 지정한 위치에 새로운 코드 삽입
* save                      : ./user/ 경로에 텍스트 파일로 영구 저장
* exit 또는 :q               : 에디터를 종료하고 일반 vShell 모드로 복귀

2. vShell 핵심 명령어 요약
-------------------------------------------------------------------------------
* create <alias>            : 의미셀 동적 생성
* delete <alias>            : 의미셀 데이터 초기화 및 가비지 컬렉팅
* <alias>.energy = <값>     : 셀 에너지 대입 및 연산 (=, +=, -=, *=, /=)
* <alias>.Pth = <값>        : 양수 발화 임계값 설정
* <alias>.Nth = <값>        : 음수 발화 임계값 설정
* <alias>.exec = true/false : 발화 시 vCPU 스크립트 트리거 여부 설정
* <alias>.active = true     : 임계값 무시하고 강제 발화 비트 On/Off
* <A>.(<B>) = <가중치>       : A에서 B로 가는 시냅스 가중치 제어
* link_fact <A> <B> [W]     : 'A는 B다' 라는 팩트 논리 관계 바인딩
* unlink <A> <B>            : A와 B 사이의 시냅스 연결 완벽 파괴
* decay_links <A> <비율>    : A에 연결된 모든 가중치 일괄 감쇠 (망각)
* propagate [steps]         : 발화된 셀들로부터 가중치를 곱해 순방향 전파
* reset_energy_all          : 가상 메모리 망 내의 모든 잔류 에너지 청소
* fire <alias>              : 특정 셀에 에너지 1.0 주입하여 즉각 발화
* print_fired               : 현재 활성화(Firing) 상태인 모든 노드 출력
* train_token <단어>        : 단어를 비트 패턴으로 토큰셀에 반복 학습
* scan_tokens <문장>        : 초성 체인을 고속 순회하며 일치하는 토큰 발화
* link_token <단어> <셀>    : 인지된 토큰셀과 고차원 의미셀을 시냅스로 연결
* bind_script <셀> <파일>   : 특정 셀에 vCPU 가동용 확장 스크립트 바인딩
* trigger_vcpu              : 인터럽트 스캔 후 execFlag가 켜진 스크립트 순차 실행
* math <셀> = <함수> <값>   : CellMath 코어를 직접 호출해 수학 연산 적용
* sync                      : mmap 가상 메모리를 실제 SSD 파일로 강제 플러시
* wait <ms>                 : 지정된 밀리초만큼 스크립트 실행 스레드 대기
